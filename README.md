# 3346 AI Game Project

HOW TO RUN:

- Run 3346 Game.exe

HOW TO VIEW CODE OR RUN IN EDITOR:

- Download Godot 4.5.1 (https://godotengine.org/download/windows/)
- When Godot is installed, open the folder containing the godot.project file
- Press the left most play button on the top right of the editor. This will load the main scene which is the gamescene


CONTROLS:

WASD OR Arrow keys - Movement directions
R - Reload level (to help with testing)


If you have troubles running or have any questions at all, email any of us in the group and we will gladly help.
